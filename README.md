# Madhav Pediatric Dental Care

A premium, full-stack web application for **Madhav Pediatric Dental Care & Implant Center**, Nikol, Ahmedabad.

## ✨ Features

- **Modern UI/UX**: Built with Next.js 15, Tailwind CSS, and Framer Motion for smooth, high-end animations.
- **Dynamic Content**: Doctors, Treatments, and Gallery are fetched from a custom Backend API.
- **Admin Panel**: Secure dashboard for the clinic staff to manage:
  - **Appointments**: View and track patient bookings.
  - **Treatments**: Add or update dental services offered.
  - **Doctors**: Manage specialist profiles and photos.
  - **Gallery**: Upload and organize clinic photos.
  - **Clinic Settings**: Live updates for contact info and working hours.
- **Interactive Map**: One-click Google Maps integration for patients.
- **Responsive Design**: Optimized for Desktop, Tablet, and Mobile devices.
- **Fast Performance**: Next.js App Router with Skeleton loading states for a seamless experience.

## 🛠️ Technology Stack

- **Frontend**: Next.js, TypeScript, Tailwind CSS, Framer Motion.
- **Backend**: FastAPI (Python), SQLModel, SQLite.
- **Authentication**: JWT-based secure Admin login.

## 🚀 Getting Started

### 1. Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 2. Frontend Setup
```bash
npm install
npm run dev
```
